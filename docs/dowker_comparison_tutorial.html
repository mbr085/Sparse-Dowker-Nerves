Comparison of sparsification strategies¶

In this tutorial we describe the use of the python package dowker_comparison to calculate the persistent homology of point clouds. This is meant as a reference to compare the different strategies and examples in Sparse Dowker Nerves (SDN) and Sparse Filtered Nerves (SFN). The recommended sparsification strategy is implemented in dowker_homology.

In this package we provide classes for calculating persistent homology using different truncation and sparsification methods. We have intrinsic and ambient classes for most methods. All classes are follow the naming scheme Dowker_Intrinsic_Truncation_Sparsification or Dowker_Ambient_Truncation_Sparsification and take the same arguments as their counterparts in dowker_homology.

Truncation methods¶

We have implemented truncation methods described by Cavanna, Jahanseir and Sheehy in A Geometric Perspective on Sparse Filtrations and Definition 6 in Sparse Dowker Nerves, which will be termed 'Sheehy'.

Restriction methods¶

The two restriction methods implemented here come from Sparse Dowker Nerves and are termed 'Sheehy' for the method described in Proposition 1 and 'Parent' for the method described in Definition 8.

Clifford torus¶

We start with calculating ech persistent homology of a point cloud in 4. First we import the relevant python packages, set a random seed and generate the sample data used for the examples below. The data we use are 100 points on the Clifford torus.

# generate data
def clifford_torus(N):
    x = np.linspace(0, 2*np.pi, num=N, endpoint=False).reshape(N,1)
    y = np.sqrt(N)*x
    return np.hstack((np.cos(x), np.sin(x), np.cos(y), np.sin(y)))

coords = clifford_torus(100)

In this case the parent restriction was larger than the sheehy restriction for the intrinsic case and smaller for the ambient case. Note that the two restrictions result in homotopy equivalent nerves. The truncation and restriction methods presented in Sparse Filtered Nerves are considerably smaller, but result in different persistent homology with the same interleaving guarantees. It looks like the ambient version using Sheehy truncation is closeer to the original persistence diagram than the version in Sparse Filtered Nerves, but results in much larger nerves. So the fact that the Sheehy truncation is closer to the original persistence diagram comes at the cost of taking much more time without any additional guaranteed benefit.
